# Thrive Sports Therapy Website

This repository contains the full landing page for Rami Sports Med.

## Features
- About section with professional bio
- Services section for sports injury rehab, performance training, preventative care
- Fully interactive booking system integrated with Google Calendar
- Contact section with map
- Responsive design for mobile and desktop

## Setup / Customization
1. Replace `CLIENT_ID`, `API_KEY`, `CALENDAR_ID` in `script.js` with your Google Calendar API credentials.
2. Replace brand name, logo, and images in `assets/images/`.
3. Customize colors and styling in `style.css`.
4. Commit and push changes.

## Deployment
Use GitHub Pages to deploy:
1. Go to Settings → Pages → Select main branch → root folder.
2. The live site will appear at:
   `https://yourusername.github.io/rami-sportsmed-site/
